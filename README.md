Group Chat Application

Real-time anonymous group chat application built with Node.js, Socket.IO, and MySQL.

## Features
-  Real-time messaging
-  Anonymous mode toggle
-  iPhone-style UI design
-  Message history
-  Multiple users support


### Prerequisites
- Node.js
- MySQL
